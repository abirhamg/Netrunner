# Netrunner
This repository provides instructions for installing Netrunner Linux using VMware Workstation. It includes step-by-step guidance to set up a virtual machine, configure system resources, and run Netrunner smoothly on your host system without affecting your main operating system.
Netrunner on VMware Workstation
Quick guide to install Netrunner Linux in VMware.
Prerequisites
VMware Workstation
Netrunner ISO
2 CPU cores, 4 GB RAM, 20 GB disk
Steps
Create VM → New VM → Typical → ISO file
Configure → Name: Netrunner, Disk: 20 GB+, RAM: 4–8 GB, CPU: 2 cores, Network: NAT/Bridged
Install → Boot VM, follow installer, set language/timezone/user, Guided disk
Post-Install → Remove ISO, install VMware Tools, adjust display/folders
