# Netrunner
This repository provides instructions for installing Netrunner Linux using VMware Workstation. It includes step-by-step guidance to set up a virtual machine, configure system resources, and run Netrunner smoothly on your host system without affecting your main operating system.
Netrunner Installation on VMware Workstation
A simple guide to install Netrunner Linux in VMware Workstation.
Prerequisites
VMware Workstation installed
Netrunner ISO downloaded
Minimum: 2 CPU cores, 4 GB RAM, 20 GB disk
Steps
Create VM
Open VMware → New Virtual Machine → Typical → ISO file.
Configure VM
Name: Netrunner
Disk: 20 GB+
RAM: 4–8 GB, CPU: 2 cores
Network: NAT or Bridged
Install Netrunner
Boot VM → follow installer
Set language, timezone, user account
Partition: Guided – use entire disk
Reboot when done
Post-Installation
Remove ISO from CD/DVD drive
Install VMware Tools
Adjust display, shared folders, snapshots
